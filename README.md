### 👨‍💻 About Me

I'm an AI engineer focused on the layer most demos skip — taking models from "works in a notebook" to "works reliably in production." I care less about the model itself and more about everything around it: orchestration, evaluation, and recovery.

- 🤖 **Focused on** LLMs, AI agents, and machine learning
- 🛠️ **Currently building** toward agent reliability — durable, resumable workflows that recover from failure instead of restarting from zero
- 🌱 **Currently learning** TensorFlow
- 👯 **Looking to collaborate on** AI agents
- 💬 **Ask me about** AI, LLMs, agents, Python, and machine learning
- 📫 **Reach me at** [sydafu.work@gmail.com](mailto:sydafu.work@gmail.com)
